##<a name="快乘">快速乘</a>
>内置的默认乘,时间复杂度log(n),原理： 如果b=(101)_2,则b*a=1*a*2^2+0*a*2^1+1*a*2^0;

```java
//
long fast_nul(long a,long b){
  long res=0;
  while(b!=0){
    if((b&1)==1){
      res+=a;
     }
     a*=2;
     b>>>=1;
  }
  return res;
}
```
