## <a name='快速乘'>快速乘</a>
>内置的默认乘,时间复杂度log(n),原理： 如果 b=(101)<sub>2</sub>,则b*a=1&times;a&times;2<sup>2</sup>+0&times;a&times;2<sup>1</sup>+1&times;a&times;2<sup>0</sup>;

```java
long fast_nul(long a,long b){
  long res=0;
  while(b!=0){
    if((b&1)==1){
      res+=a;
     }
     a*=2;
     b>>>=1;
  }
  return res;
}
```
