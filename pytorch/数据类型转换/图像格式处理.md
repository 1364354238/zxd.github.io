# 图片的读取与显示
---

## 读取图像的方式
```python
//图片路径
img_path='E:/CoCo/test2017/000000000001.jpg'
```

### 1. CV

读取的是numpy.ndarray()格式,BGR顺序

```python
imoprt cv2

#读取图片
img_cv=cv2.imread(img_path)

#创建一个窗口界面
cv2.namedWindow("Image")

#在窗口显示图像
cv2.imshow(img_cv)

#设置结束按键，
cv2.waitKey(0)

#释放窗口
cv2.destoryAllWindows()
```
